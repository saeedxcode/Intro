# Personal Information App

A simple C# console application that collects and displays personal information with BMI calculation.

## Features

- Collects personal information (name, age, height, weight, city, job)
- Calculates BMI and displays health status
- Shows age category and approximate birth year
- Input validation to prevent errors
- Clean console output

## How to Run

1. Make sure you have .NET SDK installed
2. Clone or download this project
3. Open terminal in project folder
4. Run:
   ```bash
   dotnet run
